## Что такое JARVIS

Telegram-бот на твоём компьютере, подключённый к Claude AI (Anthropic).
Работает 24/7, запускается автоматически при включении, не требует браузера или приложений.

Ты пишешь ему в Telegram — он отвечает, помогает, выполняет задачи.

## Установка — одна команда

Открой терминал и вставь:

```bash
curl -s https://raw.githubusercontent.com/sssviridov1-ops/jarvis-child/main/install.sh | bash
```

**Требования:**
- Mac (macOS) или Linux
- Python 3 — скачать на [python.org](https://python.org) если нет
- Интернет

## После запуска установщика

Тебе нужно будет заполнить 3 поля в файле `~/jarvis_bot/bot/config.py`:

### 1. TELEGRAM_TOKEN
Создай своего бота:
1. Открой Telegram, найди [@BotFather](https://t.me/BotFather)
2. Напиши `/newbot`
3. Придумай имя и username для бота
4. Скопируй токен (выглядит как `1234567890:ABCdef...`)

### 2. ANTHROPIC_KEY
1. Зайди на [console.anthropic.com](https://console.anthropic.com)
2. Зарегистрируйся
3. Раздел API Keys → Create Key
4. Скопируй ключ (начинается с `sk-ant-...`)

### 3. CHAT_ID
1. Найди в Telegram [@userinfobot](https://t.me/userinfobot)
2. Напиши ему `/start`
3. Скопируй число которое он пришлёт (твой ID)

## Команды

```
/помощь   — список всех команд
/new      — сбросить историю разговора
/opus     — умная модель (сложные задачи)

```

## Автозапуск

После установки бот:

- **Linux** — запускается как systemd сервис

Перезагрузил компьютер — бот снова работает. Ничего делать не нужно.

## Частые вопросы

**Бот не отвечает?**
Проверь что бот запущен:
```bash
ps aux | grep claude_bot
```
Если не запущен — запусти вручную:
```bash
cd ~/jarvis_bot && python3 bot/claude_bot.py &
```

**Закончились деньги на Claude?**
Пополни баланс на [console.anthropic.com](https://console.anthropic.com) → Billing

**Как остановить бота?**
```bash
pkill -f claude_bot.py
```

**Как обновить до новой версии?**
```bash
cd ~/jarvis_bot && git pull && pkill -f claude_bot.py
```
Бот перезапустится сам через LaunchAgent/systemd.
